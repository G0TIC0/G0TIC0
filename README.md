## 🚀 Projeto em Destaque

### 📋 [Martins Orçamentos — AI Budget System](https://github.com/G0TIC0/martins-server-rs)
> Sistema de criação e gestão de orçamentos com **geração assistida por Google Gemini**.

- 🤖 IA integrada via Google Gemini API
- 💾 Banco de dados com Supabase (PostgreSQL + migrações)
- ⚡ Frontend em TypeScript + Vite
- ☁️ Deploy serverless no Vercel

🔗 **[Ver demo ao vivo →](https://martins-server-rs.vercel.app)**
